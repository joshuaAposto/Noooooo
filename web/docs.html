Basic Usage

Making Requests

Simply make an HTTP request (usually GET, but check specific endpoint details) to the API paths listed. No keys or complex headers needed.

Example (JavaScript Fetch):

fetch('https://zen-api.gleeze.com/api/example/users')
  .then(response => {
    if (!response.ok) {
      throw new Error(`HTTP error! Status: ${response.status}`);
    }
    return response.json(); // or response.text(), response.blob() etc.
  })
  .then(data => {
    console.log('API Data:', data);
    // Process your data here
  })
  .catch(error => {
    console.error('Fetch error:', error);
  });

Handling Responses

Most endpoints return JSON. Image-based endpoints typically return `image/png` or `image/jpeg`. Always check the `Content-Type` response header. Use the interactive modal here to easily see response types and data structures.
